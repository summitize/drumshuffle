# Changelog

All notable changes to Drumshuffle are documented here.

---

## [2026-05-27] – UI Fixes & Feedback System

### Added
- **Socials tab** with a "New" badge in the navbar — Google Wallet-style social passes with customisable colours, URLs, usernames, QR codes, and dynamic fields.
- **Admin Panel** (accessed via secret `ved` console trigger) to add/delete social media cards.
- Preset vector icons and ability to upload custom SVG icons for social passes.
- **Feedback Panel** — user submissions from the homepage contact form now appear in the CMS Admin feedback tab.
- Stable **antigravity easter-egg** with a proper "Close / Kill Process" button to end the animation cleanly.

### Fixed
- Overlapping navbar in CMS — applied global `scroll-padding-top` to prevent content being hidden under the fixed nav.
- Typo "wensite" → **website** in the hero section text.

### Changed
- Various UI polish: glassmorphic cards, gradient presets, micro-animations, improved responsiveness.

---

## [2026-05-14] – Sound Store, Sheets & More

### Added
- **Sound Store** navigation link in the Navbar.
- New curated sample packs: *Infinite*, *Nightmare Fantasia*, *Let's Go to Space*, and *377 Drum Loops*.
- Subscription milestone badges: 2k, 3k, 4k, 5k, 10k, 50k, 100k, 1M, 2M, 5M with balanced grid layout.
- Extensive legendary bands & songs database in the Sheets library, all routed to Songsterr search.
- Tech setup gear grid updated with Alesis Max, OnePlus 5, iRig 1, tripod, guitar cable, phone holder — direct buy links included.
- State-driven Easter Egg info modal triggered by clicking the about-page profile image 12 times.
- Founder avatar replaced with Singapore landscape photo on the About page.
